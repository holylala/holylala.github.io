代理模式定义

为其他对象提供一种代理以控制对这个对象的访问。
代理对象起到中介作用，可去掉功能服务或者增加额外的服务。

常见的几种代理模式

  • 远程代理

为不同地理的对象提供局域网代表对象

  • 虚拟代理

根据需要将资源消耗很大的对象进行延迟,真正需要的时候进行创建

  • 保护代理

权限控制

  • 智能引用代理

提供额外功能

智能引用代理的实现

  • 静态代理

代理和被代理对象在代理之前是确定的。他们都实现相同的接口或者继承相同的抽象类。
